
#  TaskFlow - Frontend

TaskFlow is a modern task management web application built using **Next.js**, **TypeScript**, and **Tailwind CSS**.

It allows users to:

- Create tasks
- Update tasks
- Delete tasks
- Toggle task status (Pending / Completed)
- Secure authentication with JWT
